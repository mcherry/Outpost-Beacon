Privacy Policy

Effective date: May 22, 2026

Outpost Beacon is a macOS menu bar application developed by Mike Cherry. This policy explains what data the app handles and how it is stored.

Data Collection

Outpost Beacon does not collect, transmit, or share any personal data. The app has no user accounts, no analytics, no telemetry, and no advertising.

Data Stored on Your Device

All data created by Outpost Beacon stays on your Mac:

  • Preferences — settings such as your refresh interval and notification preferences are stored in the app's local UserDefaults container.
  • Uptime history — if you enable uptime logging, service health records and response times are stored in a local SQLite database at ~/Library/Application Support/Outpost Beacon/uptime.sqlite.
  • Check scripts — your custom JavaScript check scripts are stored locally at ~/Library/Application Support/Outpost Beacon/checks/.

None of this data is sent to the developer or any third party.

Network Activity

Outpost Beacon makes outbound network requests in two situations:

  • Service health checks — the app connects to URLs and hosts that you configure in your check scripts (e.g., status pages, HTTP endpoints, TCP ports). These requests are made directly from your device; no proxy or intermediary server is involved.
  • Update checks — if automatic update checking is enabled, the app periodically fetches a small version file from GitHub (raw.githubusercontent.com) to determine whether a newer release is available. No identifying information is sent with this request beyond what the operating system includes by default.

The app does not operate any servers and does not receive any data from your device.

Third-Party Services

Outpost Beacon does not integrate any third-party SDKs, analytics frameworks, or advertising networks. The only third-party service contacted is GitHub, solely for the optional update check described above.

macOS Permissions

The app runs in the macOS App Sandbox and requests only:

  • Outgoing network connections — required to perform health checks and update checks.
  • User-selected file access — required to export uptime reports (CSV/PDF) to a location you choose via a save dialog.

The app may also request permission to deliver macOS notifications if you enable status change alerts.

Children's Privacy

Outpost Beacon is a general-purpose utility and does not knowingly collect any information from anyone, including children under 13.

Changes to This Policy

If this policy is updated, the revised version will be posted at this URL with a new effective date. Because the app collects no data, material changes are unlikely.
